# NAI_pjatk7sem

## How to run 
Install python 3 from https://www.python.org/downloads/ or a packet manager of your choice.
Run 
`pip install numpy`,
`pip install skfuzzy`,
`pip install pygame_widgets`,
`pip install pygame`
then run 
`python ./fuzzy.py`
to start the program.

## How to use
slide Temperature, tdp, and load sliders to adjust parameters
